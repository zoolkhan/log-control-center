# Log Control Center by OH8XAT v1.3 (Windows Edition)

A real-time dual-source telemetry dashboard and ADIF log merger for Amateur Radio.

## 🚀 Windows Quick Start

### Option A: Standalone Executable (Recommended)
**No installation required!**  
1. [**Download LogControlCenterBridge_v1.3.exe**](https://github.com/zoolkhan/log-control-center/releases/download/v1.3/LogControlCenterBridge_v1.3.exe)  
2. Double-click the EXE to launch the Control Panel.  
3. Click **LAUNCH BROWSER** to open the bridge.

---

### Option B: Python Source Installation
If you prefer running from source or want to develop/modify the code:
1. **Install Python:** Download and install Python 3.x from [python.org](https://www.python.org/downloads/). (Ensure "Add Python to PATH" is checked).
2. **Run Installer:** Double-click `install.bat`.
3. **Launch App:** Double-click `run.bat`.
4. **Open Browser:** Go to [http://localhost:5000](http://localhost:5000).

---

## ⚙️ Configuration
1. Click the red **CONFIG** button in the web header.
2. Enter your Windows paths using either `\` or `/`.
   - Example: `C:\Users\Timo\Documents\Log1.adi`
   - Example VarAC Log: `C:\Users\Timo\Documents\VarAC\VarAC.log`
3. Click **SAVE & RELOAD**.
